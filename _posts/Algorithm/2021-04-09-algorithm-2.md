## 풀이과정
---

```python
def solution(phone_book):
    phone_book.sort()
    for i in range(len(phone_book)-1):
        if phone_book[i+1].startswith(phone_book[i]):
            return False  
    return True
```

해시문제인데 그냥 풀려버렸네요..
전화번호부를 Sort 하게되면 문자열이기 때문에 앞 문자가 유니코드상으로 빠른 순서대로 정렬이됩니다.

따라서 바로 뒤에 있는 것만 앞의 것으로 시작하는지(접두어) 확인하면 됩니다.
